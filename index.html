Preguntas a Resolver

1. ¿Cuál es la diferencia entre Internet y la World Wide Web?

El internet es una red de computadoras conectadas entre sí en todo el mundo, mientras que la WWW es una enorme colección de páginas alojadas en esa red.

2. ¿Cuáles son las partes de un URL?

  • Esquema (Protocolo)
  • Subdominio
  • Dominio de segundo nivel
  • Dominio de nivel superior
  • Subdirectorio
  • Puerto
  • Ruta
  • Consulta
  • Parámetros
  • Fragmento

3. ¿Cuál es el propósito de los métodos HTTP?

Los métodos HTTP permiten interactuar con un servidor. GET obtiene datos, HEAD obtiene solo los encabezados, POST envía datos, PUT reemplaza completamente un recurso y PATCH lo actualiza parcialmente.

4. ¿Qué método HTTP se debe utilizar al enviar un formulario HTML?

El método adecuado es POST, ya que envía datos al servidor.

5. ¿Qué método HTTP se utiliza al acceder a una página web mediante un navegador?

Se utiliza el método GET, ya que solicita recursos como páginas web, imágenes o archivos.

6. ¿Qué significa un código de respuesta HTTP 200?

Significa que la solicitud fue exitosa y el servidor devolvió la respuesta sin errores.

7. ¿Es responsabilidad del desarrollador corregir un error 404?

Sí, ya que indica que un recurso no fue encontrado en el servidor y debe ser corregido.

8. ¿Es responsabilidad del desarrollador corregir un error 500?

Sí, porque el error 500 es causado por fallos en el servidor y no por el usuario.

9. ¿Qué significa que un atributo HTML5 esté depreciado?

Significa que ya no es recomendado en el estándar HTML5. Ejemplos: <center>, <font>, <big>.

10. Diferencias principales entre HTML4 y HTML5

HTML5 simplifica la estructura, introduce etiquetas semánticas como <article> y mejora la compatibilidad con CSS3 y ECMAScript 6.

11. Componentes de estructura y estilo en una tabla

Estructura: <table>, <tr>, <td>, <th>. Estilos: border, padding, margin, background-color.

12. Principales controles de una forma HTML5

Text, email, password, number, color, checkbox, file, submit.

13. ¿Qué tanto soporte HTML5 tiene el navegador Opera?

Opera tiene un soporte muy alto para HTML5, incluyendo las últimas versiones.
