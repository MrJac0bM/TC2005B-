Preguntas y Respuestas

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje de programación orientado a objetos utilizado en aplicaciones empresariales y móviles, mientras que JavaScript es un lenguaje de scripting principalmente usado en desarrollo web para interactividad en el navegador.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5)

  • getFullYear() - Devuelve el año actual.
  • getMonth() - Devuelve el mes (0-11).
  • getDate() - Devuelve el día del mes.
  • getHours() - Devuelve la hora.
  • getMinutes() - Devuelve los minutos.

¿Qué métodos tienen los arreglos? (Menciona al menos 5)

  • push() - Agrega un elemento al final del arreglo.
  • pop() - Elimina el último elemento del arreglo.
  • shift() - Elimina el primer elemento del arreglo.
  • unshift() - Agrega un elemento al inicio del arreglo.
  • forEach() - Ejecuta una función por cada elemento del arreglo.

¿Cómo se declara una variable con alcance local dentro de una función?

Se puede declarar utilizando let o const dentro de una función. Por ejemplo:

function miFuncion() {
    let variableLocal = "Soy local";
    console.log(variableLocal);
}

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

El uso de variables globales puede causar conflictos si varias funciones modifican la misma variable, lo que puede generar errores difíciles de rastrear y problemas de mantenimiento en el código.
